# Pharmacy_management_system

## Getting Started

To run the program, follow these steps:

1. Compile the necessary Java files:

    ```bash
    javac connect.java 
    javac start.java 
    javac admin.java 
    javac employee.java 
    javac clearCMD.java 
    javac authentication.java
    ```

2. Run the main program:

    ```bash
    java mini
    ```
